Rawr Documentation

Rawr is a program for helping optimize the performance of World of Warcraft characters. It's designed to be simple and easy to use, but also to have advanced features available if you choose to use them. We're actively working on filling in more and more documentation here.

Before posting an Issue in the Issue Tracker, or opening a new Thread in Discussions, please refer to our Posting Guidelines, and then to our Frequently Asked Questions
