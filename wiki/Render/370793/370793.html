As you can see you can select 3 different levels of warning sounds, for Minor upgrades, Upgrades and Major upgrades. There are built in sounds that come with the addon or if you have the SharedMedia addon installed you can even setup your own custom sounds using your own MP3 files.

Also when configuring the sounds you can set the threshold at which the sound is played. So for minor upgrade the default threshold is between 0 and 10% upgrade, for a standard upgrade it's between 10% and 20% and a major upgrade is over 20%. You can change these values to suit your own personal tastes. If you have the warning frame enabled the addon will also show a visual warning and tell you the percentage upgrade that the item is.
