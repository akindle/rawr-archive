Rawr Addon Configuration

You can access the Rawr Addon configuration menu in game by pressing ESC to bring up the in game menu then clicking Interface and selecting the Addons Tab and scrolling down to Rawr. Alternatively typing /rawr config should bring up the menu. Please note that if you have many addons this may not take you directly to the Rawr config, I believe this is a bug with the Blizzard code.
